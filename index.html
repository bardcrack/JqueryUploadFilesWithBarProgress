Jqueryuploadfiles

Es un sencillo script para subir archivos de imágenes a nuestro servidor, mientras nos muestra una barra de progreso junto con su porcentaje de subida. Fácil de configurar y utilizar. Realizado utilizando Jquery, Php y Ajax. [It is a simple script to upload images to our server, while shows a progress bar with percentage rise. Easy to configure and use. Made using Jquery, Php and Ajax.]

Bienvenido.

En vista de la necesidad de incorporar un script para la subida de archivos de imagen o de cualquier otro tipo a nuestro servidor,

de una manera elegante y sin tener que refrescar la pagina.

He desarrollado este script para facilitar la subida de archivos a un servidor en este caso son de tipo de imagenes.

Es muy sencillo de utilizar y configurar, adapatandolo a sus propias necesidades, pueden modificar el tamaño maximo del archivo permitido para

subir, asi como el tipo de archivos permitido.
